## 📋 Описание

Проект содержит два взаимосвязанных документа:

### 📖 Исследование (`index.html`)
- Полный каталог из 9 подвоев с генетическими формулами
- Анализ устойчивости по родительским видам
- Региональные рекомендации для 3 субъектов РФ
- Агрономические таблицы и памятки
- 🔗 **Кнопка перехода к калькулятору**

### 🧮 Калькулятор (`calculator.html`)
- Интерактивный выбор по 7 типам почв и 2 климатическим зонам
- Автоматическая блокировка несовместимых вариантов
- Мгновенные рекомендации с пояснениями
- 🔗 **Кнопка возврата к исследованию**
