🔹 Практические правила выбора по генетике:

📌 Если в генетике подвоя есть Рипариа → ожидайте сниженной устойчивости к засухе и извести. Такой подвой (101-14) лучше сажать на влажных, некислых, некраснозёмных почвах.

📌 Кобер 5ББ и СО4 — исключение: при наличии Рипариа в формуле (Берландиери × Рипариа) доминируют гены Берландиери, поэтому подвой показывает хорошую универсальность и подходит для широкого диапазона почв.

📌 Если в генетике преобладает Берландиери → подвой будет устойчив к извести и засухе (Феркаль, 41Б, 1103 Полсен, Рюгжери 140).

📌 Комбинация Берландиери × Рупестрис → «золотой стандарт» для засушливых и каменистых участков с умеренным содержанием извести.

📌 Если в генетике есть Амурский виноград → подвой будет максимально морозостойким и адаптированным к континентальному климату (Андрос).

🔗 Часть 4. Аффинитет прививок: роль Виниферы в совместимости

📌 Аффинитет прививки — это степень биологической совместимости подвоя и привоя, которая проявляется в:
• Равномерном срастании камбиальных слоёв
• Отсутствии утолщений, наплывов или перетяжек в месте спайки
• Сбалансированном обмене водой и питательными веществами
• Долговечности куста без признаков «отторжения»

⚠️ Чистые американские подвои (без Виниферы)Могут требовать подбора совместимых привоев

ФакторОбъяснение

📌 Золотое правило:
Рюгжери 140 и 1103 Полсен — не для «ленивого» виноградника. Только для южных регионов! Они раскрывают потенциал на бедных почвах, при грамотной агротехнике и с сильнорослыми сортами. На плодородных чернозёмах без контроля они могут «уйти в ботву».

🗺️ Часть 6. Итоговая памятка по выбору подвоя

🔸 Почва известковая + засуха → 1103 Полсен, Рюгжери 140, 41Б 🔸 Почва каменистая + сухая → Рюгжери 140, 1103 Полсен 🔸 Почва лёгкая + влажная + прохладный климат → 101-14 (с укрытием, контроль извести) 🔸 Универсальный вариант для суглинков → Кобер 5ББ, СО4 🔸 Нужен компактный куст (плотная посадка) + раннее созревание → 41Б (на известковой почве) 🔸 Капризный сорт / важна совместимость → Гены Виниферы: Феркаль, 41Б 🔸 Северные районы / укрытие / риск промерзания → Андрос, 101-14 🔸 Максимальная морозоустойчивость корней (-30°С и ниже) → Андрос 🔸 Ускорение созревания привоя → 101-14 🔸 Раннее вступление в плодоношение + оптимальный рост → СО4 🔸 Сильнорослый сорт + засушливый склон (юг) → Рюгжери 140, 1103 Полсен 🔸 Песчаные почвы + прохладный климат → Гравесак 🔸 Континентальный климат + резкие перепады температур → Андрос 🔸 Феркаль, 1103 Полсен, Кобер 5ББ, Рюгжери 140 — не для богатых черноземов 🔸 Самая низкая морозоустойчивость корней → 41Б (требует укрытия или замены)

💡 Ключевые рекомендации

✅ Перед закладкой виноградника:

  1. Проведите анализ почвы на содержание активной извести, солей и гранулометрический состав
  2. Определите уровень грунтовых вод — для сильнорослых подвоев нужен глубокий горизонт; близкое залегание вод вызывает гнили корней, создаёт анаэробные условия и лишает корни возможности нормального питания
  3. Оцените риск весенних заморозков 101-14 и зимнего промерзания почвы — для 41Б это критично
  4. Запланируйте систему орошения — особенно для песчаных почв для подвоя 101-14, для СО4 на склонах с низким уровнем грунтовых вод

⚠️ Типичные ошибки:

  • ❌ Выбор сильнорослых подвоев (1103 Полсен, Рюгжери 140) для плодородных чернозёмов без контроля роста
  • ❌ Посадка 101-14 на карбонатных почвах без контроля извести (риск хлороза)
  • ❌ Игнорирование потребности в поливе для СО4 и 101-14 на засушливых участках
  • ❌ Посадка 41Б в зонах с промерзанием почвы ниже -12°C без укрытия

📌 Золотое правило:
Выбирайте тип корневой системы исходя из глубины залегания грунтовых вод и уровня увлажнения почвы:
• Глубокие воды + засуха → ныряющая система
• Средние воды + умеренная влага → полуныряющая система
• Высокие воды + влажность → стелющаяся система (с осторожностью!)
